### python 相关命令
```
# 输出相关依赖
python freeze > requirments.txt

# 下载相关依赖
pip install -r requirements.txt

# 创建虚拟环境
python -m venv NS3venv

# 激活虚拟环境 Linux
source NS3venv/bin/activate

# 激活虚拟环境 Windows
.\NS3venv\Scripts\activate

# 退出虚拟环境
deactivate


# python永久换源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```
